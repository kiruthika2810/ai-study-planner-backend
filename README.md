# AI Study Planner Backend

A backend application for managing study tasks using Node.js, Express.js, MongoDB, and REST APIs.

## Features

- Create Study Tasks
- Get All Tasks
- Get Single Task
- MongoDB Database Integration
- REST API Architecture
- Render Deployment
- Postman API Testing

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman
- Render
- GitHub

## API Endpoints

### Create Task

POST /create-task

### Get All Tasks

GET /tasks

### Get Single Task

GET /task/:id
